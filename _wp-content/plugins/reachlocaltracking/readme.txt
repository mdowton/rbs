Places ReachLocal's asynchronous tracking code on all pages.

== Description ==

This plugin adds the required javascript code on all pages in order to track analytics and enable other features for ReachLocal's Edge product.

For more information visit:

[ReachLocal](http://www.reachlocal.com)

== Installation ==

1. Upload `reachlocaltracking` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the reachlocal_tracking ID from Edge to the plugin settings (Admin > Settings > ReachLocal Capture)
